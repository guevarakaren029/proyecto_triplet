Proyecto de aplicación web con entorno virtual de Python.
# Cuadro de Mandos de Vehículos

Esta aplicación web permite visualizar y explorar datos de anuncios de venta de vehículos en Estados Unidos. Está construida con **Streamlit** y utiliza **Plotly Express** para generar gráficos interactivos.

## Funcionalidades

- Visualizar la **distribución de precios** de los vehículos.
- Mostrar la cantidad de vehículos por **tipo** y por **tipo de combustible**.
- Visualizar un **histograma del odómetro** de los vehículos mediante una casilla de verificación.
- Generar un **gráfico de dispersión precio vs odómetro** mediante una casilla de verificación.
